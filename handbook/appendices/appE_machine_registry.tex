\section{Adding a New Machine}

To add Machine 6 (example: Band Saw):

\begin{enumerate}[leftmargin=1.5em]
    \item Add to the comment table in \texttt{config.h}: \texttt{// 6 Band Saw 25min 5 Zone E}
    \item Add to \texttt{seed\_machines.py} MACHINES list: \texttt{\{"id": "6", "name": "Band Saw", ...\}}
    \item Add to \texttt{kiosk\_station/config.py} MACHINE\_IDS: \texttt{6: "Band Saw"}
    \item Run \texttt{python3 scripts/seed\_machines.py} to create the Firestore document
    \item Assemble, validate, and provision a new node with \texttt{MACHINE\_ID 6}
    \item Update user permissions for students who need access to the new machine and re-issue their cards
\end{enumerate}
